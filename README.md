
## Folders
 - setting
  set env variable
 - ftx
  ftx api wrapper
 - read-order-book-py
   read orderbook via ftx websocket api
 - twitter_search
   twitter recent reseach api

## How to use 
If you use `poetry` ,run
` poetry run python file.py`

For convenience,
```
alias po='poetry run'
alias pp='poetry run python'

function pdev () {
  poetry add -D black flake8 pytest
}
```
